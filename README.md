# IPWA01-01-
Programmierung von Webanwendungsoberflächen - Aufgabenstellung 1

Diese Webanwendung wurde im Rahmen der Fallstudie IPWA01-01 entwickelt.
Sie stellt fiktive CO₂-Emissionsdaten von Ländern und Unternehmen dar und
ermöglicht eine Filterung und Sortierung der Daten.

## Technologien
- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (Vanilla)

## Funktionen
- Tabelle mit CO₂-Emissionsdaten
- Filter nach Staat und Unternehmen
- Sortierung nach Spalten
- Responsive Design (Desktop, Tablet, Smartphone)
- Unterstützung von LTR/RTL-Schriftkulturen
- Schutz vor Code-Injection
